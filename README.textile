h1. Projet OAIHarvester2

Ce projet a pour but de fournir une implémentation Open Source du protocole de moissonnage OAI-PMH.

h2. Historique

Ce projet est un fork du projet "OAIHarvester2":http://www.oclc.org/research/activities/past/orprojects//harvester2/harvester2.htm développé par l'"OCLC":http://www.oclc.org/ dans sa dernière version 0.1.12.

Par rapport au projet original, celui-ci apporte les modifications suivantes :
* Gestion de configuration basée sur "Git":http://www.git-scm.com/
* Renommage du package "ORG.oclc" en "org.oclc" pour respecter les conventions java
* Utilisation de Maven comme gestionnaire de build
